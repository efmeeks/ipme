## Get

* Download the script
  * `curl -L https://raw.githubusercontent.com/efmks/ipme/master/ipme.sh -o ipme.sh`
* Make it executable
  * `chmod +x ipme.sh`
* Place somewhere in your `$PATH`
  * `mv ipme.sh /usr/local/bin/ipme`
* Check with `which` or `type`
    ```text
    $ which ipme
    /usr/local/bin/ipme
    ```

    ```text
    $ type ipme
    ipme is hashed (/Users/eric/bin/ipme)
    ```
